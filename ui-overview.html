Changes in This Update

🏥
Medical Survey Redesigned — 6 New Diseases
health-survey.html completely rebuilt. Now shows: Asthma, COPD, Tuberculosis, Bronchitis, Lung Cancer, Pneumonia + free-text "Other" field. Custom checkbox UI that avoids the generic AI-generated look — clean row cards with icons, subtle selection states, no Tailwind grid boxes.
🗺️
Minimap Mode — Clean Map with Only Cursor + Safe Zone Distance
map.html now detects when it's embedded as an iframe (via ?minimap=1 param or window.self !== window.top). In minimap mode: all HUD panels, legend, AQI panel, controls, attribution are hidden. Only the map tiles + your cursor marker + safe zone dot remain. A minimal overlay at the bottom shows "SAFE ZONE · 4.2 km · NE".
🤖
OxyBot — Draggable Button + API Key Fix
chatbot.js now supports drag-to-reposition. Hold and drag the bot bubble anywhere on screen — it snaps to viewport edges and the chat window follows. On release without movement, it opens/closes as normal. API key error now shows a clear in-chat notice with a link to get a free Gemini key instead of a confusing error. The hardcoded invalid key has been replaced with a placeholder.

Setup Checklist

🔑
Set Your Gemini API Key
In index.html and map.html, find window.GEMINI_API_KEY = 'YOUR_GEMINI_API_KEY' and replace with your key. Get one free at aistudio.google.com. The chatbot will show a warning inside the chat if the key is missing.
🔥
Firebase Config Already Set
firebase.js is already configured with the OxyTrace project credentials. No changes needed unless you want to use your own Firebase project.
🌐
APIs Used (No Keys Required)
Open-Meteo for live AQI data and BigDataCloud for reverse geocoding — both work without API keys. GPS location is acquired from the browser.
